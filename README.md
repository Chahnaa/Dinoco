# Dinoco

Movie Review application with a Python backend and a React + TypeScript frontend.

## Structure
- movie-review-backend: Flask API, database schema, and Docker setup
- movie-review-frontend: React app with Tailwind styling

## Setup
### Backend
1. Navigate to `movie-review-backend`
2. Create a virtual environment and install dependencies from `requirements.txt`
3. Run the app with `python app.py`

### Frontend
1. Navigate to `movie-review-frontend`
2. Install dependencies with `npm install`
3. Start the dev server with `npm start`
